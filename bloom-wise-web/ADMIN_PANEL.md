# Админ-панель

## Доступ

Админ-панель доступна только пользователям с ролью `admin`.

**Админский аккаунт по умолчанию:**
- Email: `admin@bloomwise.com`
- Password: `admin123`

## Функции админ-панели

### Управление растениями

1. **Просмотр всех растений**
   - Список всех растений в системе
   - Отображение изображения, названия, описания и цены

2. **Добавление растения**
   - Название (обязательно)
   - Описание (обязательно)
   - Цена (обязательно)
   - URL изображения (опционально)
   - Уровень ухода: Легкий / Средний / Сложный
   - Требования к освещению: Низкое / Среднее / Высокое
   - В наличии (переключатель)
   - Безопасно для животных (переключатель)

3. **Редактирование растения**
   - Изменение всех полей растения
   - Сохранение изменений

4. **Удаление растения**
   - Удаление с подтверждением
   - Автоматическое удаление связанных записей (корзина, избранное)

## API Эндпоинты

Все админские эндпоинты требуют:
- JWT токен в заголовке `Authorization: Bearer <token>`
- Роль `admin` у пользователя

### Эндпоинты:

- `GET /api/admin/plants` - Получить все растения
- `POST /api/admin/plants` - Создать растение
- `PUT /api/admin/plants/:id` - Обновить растение
- `DELETE /api/admin/plants/:id` - Удалить растение

## Навигация

Админ-панель доступна:
- Через меню навигации (для админов)
- Через профиль пользователя (ссылка "Админ-панель")
- Прямой переход на `/admin`

## Безопасность

- Все админские эндпоинты защищены middleware `requireAdmin`
- Проверка роли происходит на уровне backend
- Frontend скрывает админские функции для обычных пользователей
